# EXOPLANET HABITABILITY ANALYSIS:

This is an exploratory analysis of 4575 exoplanets from the NASA Exoplanet Archive (2021)
The dataset has 4575 rows and 23 columns, plus a header meaning it roughly has aboout 4575 exoplanets.

##KEY FINDINGS:

 - A moderate correlation (r = 0.391) between stellar mass and planet mass.
 - Confirmation of Kepler's Third Law (square of a planet's orbital period is proportional to the cube of it's distance from the parent star) across 2605, exoplanets (r = 0.995) where r = correlation.
 - Multi-Planet systems show a lower mean eccentricity value as compared to single planet systems.
 - Custom habitability index ranks HD 28185 b as the top candidate and Kepler 22 - b as the second.
